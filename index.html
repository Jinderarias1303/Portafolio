Proyectos.

Sitio Web estudio fotografico para Matrimonios

se planteaba la necesidad de crear una sitio web para un matrimonio, donde se pudieran mostrar los Trabajos realizados por el fotografo, y que los usuarios pudieran ver las fotos de la boda.

El sitio web fue desarrollado utilizando HTML, CSS, BULMA y se implementó un diseño responsivo para garantizar una experiencia de usuario óptima en dispositivos móviles y de escritorio.

El sitio web cuenta con una galería de fotos, donde los usuarios pueden ver las imágenes de la boda y un formulario de contacto para que los interesados puedan comunicarse con el fotógrafo.
